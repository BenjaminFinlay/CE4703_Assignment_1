<h1>CE4703_Assignment_1</h1> 

This is my First assignment project for computer software 3
Its a C project but we have to submit Doxygen Documentation

This is an assignment for my class CE4703 Software 2 that focuses on C and starting to program at a lower level.
The document for the project asks us to create 15 working functions, a few simple functions, array operations function and maths base functions.
The project is split into 5 modules where each contain a C file and header file.
The Header files contains the function definitions, preprocessor macros and  C files contain the function implentations.


<h2>Project Structure</h2>

<h3>Main Functions</h3>
<ul>
  <li>Main.c</li>
  <li>Main_Header.c</li>
  <li>Handles the console inputs and outputs while the main header contains the preprocessor macros</li>
</ul>

<h3>Simple Operations functions</h3>
<ul>
  <li>Simpple_Operations_Functions.c</li>
  <li>Simple_Operations_Header.c</li>
  <li>Handles the more basic functions which print all elements in the array or generates a random number</li>
</ul>

<h3>Math Opeations functions</h3>
<ul>
  <li>Matrice_Operations_Functions.c</li>
  <li>Matrice_Opeations_Header.h</li>
  <li>Handles the functions which perform mathematical operations one finds the mean average or standard deviation of the array</li>
</ul>

<h3>Matrice Operations functions</h3>
<ul>
  <li>Matric_Operations_Functions.c</li>
  <li>Matrice_Operations_Header.h</li>
  <li>Contains functions which directly modify the array such as clearing it or sorting using bubble sort</li>
</ul>

<h3>Menu functions</h3>
<ul>
  <li>Menu_Functions.c</li>
  <li>Menu_Header.h</li>
  <li>Handles the three menu options that calls the working functions in a certain order</li>
</ul>

<h3>Doxygen Documentation</h3>
<p>The project is commented in doxygen format to generate accurate documentation in HTML format with navigation</p>
